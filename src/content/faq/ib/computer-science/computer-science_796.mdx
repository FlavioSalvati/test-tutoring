Object-Oriented Programming (OOP) plays a vital role in agile methodologies by fostering code reusability and modularity.

Agile methodologies, including frameworks like Scrum and Extreme Programming (XP), prioritize adaptability and collaboration in software development. These approaches emphasize the importance of delivering functional software frequently and being responsive to changes in requirements, even late in the development process. OOP complements these agile principles by providing a structure that is both easy to modify and extend.

At its core, OOP is a programming paradigm that centers around 'objects,' which are instances of classes. These objects serve as self-contained units of code that encapsulate both data and methods. This encapsulation enhances modularity, allowing each object to be developed and tested independently. This aligns closely with the agile principle of breaking down projects into manageable, user-centric features that can be developed in short iterations, or 'sprints.'

Additionally, OOP promotes code reusability through the use of inheritance. By creating new classes based on existing ones, developers can inherit attributes and behaviors, which significantly reduces coding time and minimizes errors. This capability also enhances code readability and maintainability, aligning perfectly with the agile focus on efficiency and quality.

Refactoring involves improving the design of existing code without altering its external behavior. Abstraction allows programmers to conceal the complex details of a system, exposing only the essential features, which simplifies understanding and modification of the code. Meanwhile, polymorphism enables objects of different classes to be treated as instances of a common superclass, leading to more flexible and dynamic code.

In conclusion, the principles of encapsulation, inheritance, abstraction, and polymorphism inherent in OOP significantly bolster agile methodologies. They empower agile teams to deliver high-quality, adaptable software in a timely and efficient manner.
